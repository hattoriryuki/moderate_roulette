### サービス概要

何かで迷った時に、
ルーレットを使って、すぐに迷いを解消することのできるサービスです。

### メインのターゲットユーザー

優柔不断な方
今日の夜ご飯が決まっていない方

### ユーザーが抱える課題

優柔不断で物事を決めるのに時間がかかってしまう。

### 解決方法
このサービスを通じて、悩む時間を減らして、QOLを上げる。
